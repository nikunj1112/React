
    ```bash
    npm install
    # or

    ```

    ```bash

    # or

    ```

## ğŸ’¡ Implementation Highlights

* **Custom Hooks:** Effective use of `useEffect` and `useState` for component lifecycle and state management.
* **Unique Chat ID:** A stable `docId` is generated by sorting sender and receiver emails to maintain a single, consistent chat history in Firestore.
